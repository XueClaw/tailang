## 📄 输入：自然语言代码

```meng
# 邮箱密码登录 qwq

```python
import bcrypt

def verify_password(stored, provided):
    return bcrypt.checkpw(provided.encode(), stored.encode())
```

如果 验证成功:
    返回 令牌
否则:
    返回 错误
```

## 🔍 第一步：Lexer 词法分析

**输入**: 源代码文本  
**输出**: Token 流

### Token 序列

| # | Token | 值 | 行号 |
|---|-------|-----|------|
| 1 | Comment | `# 邮箱密码登录 qwq` | 1 |
| 2 | Newline | `\n` | 1 |
| 3 | CodeBlock | `python...` | 3-8 |
| 4 | Newline | `\n` | 8 |
| 5 | If | `如果` | 9 |
| 6 | Identifier | `验证成功` | 9 |
| 7 | Colon | `:` | 9 |
| 8 | Newline | `\n` | 9 |
| 9 | Return | `返回` | 10 |
| 10 | Identifier | `令牌` | 10 |
| 11 | Newline | `\n` | 10 |
| 12 | Else | `否则` | 11 |
| 13 | Colon | `:` | 11 |
| 14 | Newline | `\n` | 11 |
| 15 | Return | `返回` | 12 |
| 16 | Identifier | `错误` | 12 |
| 17 | Eof | - | 12 |

## 🌳 第二步：Parser 语法分析

**输入**: Token 流  
**输出**: AST (抽象语法树)

### AST 结构

```
Program
├─ CodeBlock (Python)
│   └─ "import bcrypt\ndef verify_password..."
└─ IfStmt
    ├─ condition: Identifier("验证成功")
    ├─ then_branch: BlockStmt
    │   └─ ReturnStmt
    │       └─ value: Identifier("令牌")
    └─ else_branch: BlockStmt
        └─ ReturnStmt
            └─ value: Identifier("错误")
```
